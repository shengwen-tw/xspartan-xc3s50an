# xspartan-xc3s50an

## Environment setup

0. Prerequisite

```
sudo apt install libusb-dev libftdi-dev
```

1. Install xc3sprog

```
git clone https://github.com/xtrx-sdr/xc3sprog.git

cd xc3sprog

mkdir build

cd build

cmake ..

make
```
